EXPLICAÇÕES ADICIONAIS
================================
O entregável desta disciplina consiste em dois modelos de Machine Learning e uma API REST (Flask) que os consome.

1. MODELO 1: CLASSIFICAÇÃO (Detecção de Risco)
   - Arquivo: modelo_risco.pickle
   - Notebook: 1_Notebook_Classificacao_SYMBIO.py
   - Objetivo: Classificar cargos em risco de automação (ALTO, MEDIO, BAIXO).
   - Abordagem: Usamos um dataset sintético de 111 linhas com 3 features (repetitividade, criatividade, interação humana). O modelo Árvore de Decisão foi escolhido por sua alta interpretabilidade e adequação a regras de negócio, atingindo 91.3% de acurácia.

2. MODELO 2: AGRUPAMENTO (Talentos Ocultos)
   - Arquivos: modelo_cluster.pickle, scaler_cluster.pickle
   - Notebook: 2_Notebook_Agrupamento_SYMBIO.py
   - Objetivo: Identificar perfis de funcionários (clusters) para encontrar "Talentos Ocultos" (alta performance, baixa satisfação).
   - Abordagem: Usamos o dataset "IBM HR Analytics" e o algoritmo K-Means. Um passo crucial foi a aplicação do StandardScaler (salvo em scaler_cluster.pickle) para normalizar as 6 features de perfil. O Método do Cotovelo indicou K=4 como o número ideal de clusters.

3. API REST (FLASK)
   - Arquivo: app.py
   - Descrição: A API carrega os 3 arquivos .pickle na inicialização e expõe dois endpoints principais para predição:
     - /predict/risk (para o Modelo 1)
     - /predict/cluster (para o Modelo 2)
